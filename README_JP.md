# sd-webui-reforge-spectrum (日本語版)

[English README is here](README.md)

Stable Diffusion WebUI reForge 用の拡張機能です。チェビシェフ多項式を用いた予測（Forecasting）アルゴリズムにより、UNet の計算を適宜スキップすることで画像生成を高速化します。

## 🚀 原理と概要
Spectrum Accelerator は、サンプリング工程における UNet の出力を「毎回計算するのではなく、過去の軌跡から予測する」という戦略をとります。

1.  **実計算**: 指定された間隔（Window Size）で通常の GPU 計算を行います。
2.  **階差分析**: 過去数ステップの latent の変化を分析し、次の状態をチェビシェフ多項式で近似します。
3.  **予測スキップ**: 近似された予測値を用いることで、GPU の演算（UNet）をスキップし、処理速度を劇的に向上させます。

### 期待できる効果
*   **UNet 処理速度**: 1秒あたりのステップ数（it/s）が **約2倍～2.5倍** に向上します。
*   **トータル生成時間**: VAEデコードや後処理を含めた全体時間で **20%～50% 程度の短縮** が期待できます。

## 🛠 パラメータ設定と推奨値

| パラメータ | 範囲 | 推奨値 | 説明 |
| :--- | :--- | :--- | :--- |
| **w (Blend Weight)** | 0.0 - 1.0 | **0.40** | テイラー展開とチェビシェフ予測の混合比。高いほど高速ですが、画質変化のリスクが増します。 |
| **m (Complexity)** | 1 - 8 | **3** | チェビシェフ多項式の次数。3～4が安定性と精度のバランスに優れます。 |
| **Window Size** | 1 - 10 | **2** | 何ステップごとに実計算を行うか。`2` は「1回計算・1回スキップ」を意味します。 |
| **Flex Window** | 0.0 - 2.0 | **0.20** | サンプリングが進むにつれてWindow Sizeを動的に拡大し、後半の加速を強めます。 |
| **Warmup Steps** | 0 - 20 | **1** | 開始直後の数ステップはスキップせず、正確な軌跡を記録します。 |
| **Stop Offset** | 0 - 100 | **1** | サンプリング終了直前のスキップを停止します。最後を実計算にすることで細部を整えます。 |

## 💡 推奨される使い方
画像生成の初期段階（構図確認など）では **Window Size 3** などで高速に回し、清書段階では **Window Size 2 / w 0.4** 程度に下げることで、画質と速度の最適なバランスが得られます。

## ⚠️ 注意事項
*   本拡張機能は **reForge** 環境専用に最適化されています。
*   一部のステップで計算をスキップするため、厳密には生成結果が微変化します。画質が犠牲になると感じた場合は `w` の値を下げてください。
